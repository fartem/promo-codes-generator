# promo-codes-generator

Simple CLI tool for generating promo codes.

## Usage

| Order | Argument description | Value |
| --- | --- | --- |
| 1 | Code length | Number |
| 2 | Code prefix | String |
| 3 | Codes count | Number |

**Example**

```shell
./generator 10 Promo 1
```

**Result**

```shell
Promo5hio45oh1o
```
